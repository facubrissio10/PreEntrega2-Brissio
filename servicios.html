Nuestros Programas

Programador Front-end

Un Front-end Developer es aquella persona encargada de programar el navegador de una web, es decir, es el encargado de traducir las definiciones de diseño y estilo visual realizadas en etapas previas a códigos HTML semántico.

Programador Full-Stack

El Desarrollador Full Stack es un profesional capacitado para desarrollar distintas etapas de un proyecto web. Trabaja tanto en el back-end como en el front-end.

Programador Python

Un programador Python desarrolla soluciones robustas, potentes y escalables. Los expertos en Python se desempeñan en especialidades en auge como la Inteligencia Artificial, la Ciencia de Datos y el Desarrollo Web.
